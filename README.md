# 🇻🇳 Tiếng Việt

## 📖 Giới thiệu

**Azuu Translator** là một tiện ích hỗ trợ dịch ngôn ngữ trực tiếp trên màn hình, được thiết kế đặc biệt dành cho game thủ và những người yêu thích Game/Visual Novel chưa hỗ trợ tiếng Việt. 

## ✨ Tính năng nổi bật

* 🎯 **Hai chế độ hoạt động linh hoạt:**
  * **Mode 1 (Khung dịch):** Dịch và hiển thị ra một khung riêng biệt, có lưu lịch sử.
  * **Mode 2 (Overlay):** Dịch và đè chữ tiếng Việt trực tiếp lên khung chữ gốc của game.
* ⚡ **Công nghệ nhận diện (OCR) đỉnh cao:** Tích hợp nhiều Engine siêu tốc độ như Tesseract OCR, Windows OCR (Tối ưu cho Mode 2), RapidOCR, và PaddleOCR.
* 🧠 **Hỗ trợ đa AI Dịch thuật:** Tích hợp API của Google, Bing, DeepL, Gemini, ChatGPT, Claude, DeepSeek... cho văn phong mượt mà nhất.
* 🎮 **Trải nghiệm Game thủ (Click-through):** Cho phép khóa khung dịch, làm mờ giao diện và **bấm chuột xuyên qua** để không cản trở thao tác chơi game (Phím tắt `Alt + N`).
* 🎨 **Giao diện Sakura Night:** Giao diện Dark Mode điểm xuyết hồng pastel cực kỳ dịu mắt và tối ưu trải nghiệm người dùng.

## 📥 Hướng dẫn Tải và Cài đặt

1. Truy cập vào mục **[Releases](https://github.com/AzuShark206/Azuu-Translator/releases)** ở menu bên phải.
2. Tải file `Azuu Trans.exe` mới nhất về máy.
3. Chạy file và cấp quyền nếu Windows yêu cầu.

> ⚠️ **LƯU Ý QUAN TRỌNG VỀ WINDOWS DEFENDER:**
> Vì đây là công cụ do cá nhân phát triển (không có chữ ký số doanh nghiệp) và có sử dụng công nghệ can thiệp màn hình (Overlay), Windows Defender có thể nhận diện nhầm là phần mềm lạ (False Positive). 
> **Cách khắc phục:** Bạn hãy chọn *More Info -> Run Anyway*, hoặc thêm thư mục chứa Azuu Translator vào danh sách loại trừ (Exclusions) của diệt virus nhé! 100% an toàn.

# 🇬🇧 English

## 📖 Introduction

**Azuu Translator** is an on-screen real-time translation utility specifically designed for gamers and Visual Novel enthusiasts who play games lacking their native language support.

## ✨ Key Features

* 🎯 **Two Flexible Modes:**
  * **Mode 1 (Translation Box):** Translates and displays the text in a separate window, keeping a translation history.
  * **Mode 2 (Overlay):** Translates and overlays the translated text directly onto the game's original text boxes.
* ⚡ **Cutting-edge OCR Technology:** Integrates high-speed engines like Tesseract OCR, Windows OCR (Optimized for Mode 2), RapidOCR, and PaddleOCR.
* 🧠 **Multi-AI Translation Support:** Integrates APIs from Google, Bing, DeepL, Gemini, ChatGPT, Claude, DeepSeek... for the most natural phrasing.
* 🎮 **Gamer-centric Experience (Click-through):** Allows locking the translation frame, dimming the UI, and **clicking through it** so it doesn't interfere with your gameplay (Hotkey: `Alt + N`).
* 🎨 **Sakura Night Theme:** A Dark Mode interface accented with pastel pink, highly pleasing to the eyes and optimized for user experience.

## 📥 Download & Installation

1. Go to the **[Releases](https://github.com/AzuShark206/Azuu-Translator/releases)** section on the right menu.
2. Download the latest `Azuu Trans.exe` file.
3. Run the file and grant permissions if Windows prompts you.

> ⚠️ **IMPORTANT NOTE ABOUT WINDOWS DEFENDER:**
> Because this is a personal tool (without a corporate digital signature) and uses screen overlay technology, Windows Defender might flag it as a false positive. 
> **How to fix:** Choose *More Info -> Run Anyway*, or add the folder containing Azuu Translator to your antivirus Exclusions list! It is 100% safe.

## ⌨️ Default Hotkeys

| Hotkey | Function |
| :--- | :--- |
| `Alt + Q` | Manual OCR scan on selected area |
| `Alt + 3` | Toggle Auto continuous OCR |
| `Alt + 5` | Hide / Show all UI (Boss Key) |
| `Alt + 1` | Redraw the OCR scan area |
| `Alt + N` | Lock frame & Enable Click-through |
| `Alt + X` | Clear text currently displayed on screen |

*(You can freely change these hotkeys in the Settings menu).*
