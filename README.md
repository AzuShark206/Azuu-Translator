## 📖 Giới thiệu

**Azuu Translator** là một tiện ích hỗ trợ dịch ngôn ngữ trực tiếp trên màn hình, được thiết kế đặc biệt dành cho game thủ và những người yêu thích Game/Visual Novel chưa hỗ trợ tiếng Việt. 

> 💡 **Câu chuyện phía sau:** > Ứng dụng này (hơn 6000 dòng code Python) được xây dựng **100% bằng AI** và Azuu là người "chỉ đạo". Nó là minh chứng cho việc AI có thể giúp bất kỳ ai biến ý tưởng phức tạp thành hiện thực. Do vấn đề bảo mật và chống dịch ngược (ứng dụng đã được biên dịch C nguyên khối qua Cython), Source Code xin phép được giữ đóng (Closed Source). Các bạn có thể tải bản `.exe` để trải nghiệm trực tiếp!

## ✨ Tính năng nổi bật

* 🎯 **Hai chế độ hoạt động linh hoạt:**
  * **Mode 1 (Khung dịch):** Dịch và hiển thị ra một khung riêng biệt, có lưu lịch sử.
  * **Mode 2 (Overlay):** Dịch và đè chữ tiếng Việt trực tiếp lên khung chữ gốc của game.
* ⚡ **Công nghệ nhận diện (OCR) đỉnh cao:** Tích hợp nhiều Engine siêu tốc độ như Tesseract OCR, Windows OCR (Tối ưu cho Mode 2), RapidOCR, và PaddleOCR.
* 🧠 **Hỗ trợ đa AI Dịch thuật:** Tích hợp API của Google, Bing, DeepL, Gemini, ChatGPT, Claude, DeepSeek... cho văn phong mượt mà nhất.
* 🎮 **Trải nghiệm Game thủ (Click-through):** Cho phép khóa khung dịch, làm mờ giao diện và **bấm chuột xuyên qua** để không cản trở thao tác chơi game (Phím tắt `Alt + N`).
* 🎨 **Giao diện Sakura Night:** Giao diện Dark Mode điểm xuyết hồng pastel cực kỳ dịu mắt và tối ưu trải nghiệm người dùng.

## 📥 Hướng dẫn Tải và Cài đặt

1. Truy cập vào mục **[Releases](https://github.com/AzuShark206/Azuu-Translator/releases)** ở menu bên phải.
2. Tải file `Azuu Trans.exe` mới nhất về máy.
3. Chạy file và cấp quyền nếu Windows yêu cầu.

> ⚠️ **LƯU Ý QUAN TRỌNG VỀ WINDOWS DEFENDER:**
> Vì đây là công cụ do cá nhân phát triển (không có chữ ký số doanh nghiệp) và có sử dụng công nghệ can thiệp màn hình (Overlay), Windows Defender có thể nhận diện nhầm là phần mềm lạ (False Positive). 
> **Cách khắc phục:** Bạn hãy chọn *More Info -> Run Anyway*, hoặc thêm thư mục chứa Azuu Translator vào danh sách loại trừ (Exclusions) của diệt virus nhé! 100% an toàn.

## ⌨️ Phím tắt mặc định

| Phím tắt | Chức năng |
| :--- | :--- |
| `Alt + Q` | Quét OCR thủ công vùng đã chọn |
| `Alt + 3` | Bật / Tắt Auto OCR liên tục |
| `Alt + 5` | Ẩn / Hiện toàn bộ giao diện (Boss Key) |
| `Alt + 1` | Vẽ lại vùng cần quét OCR |
| `Alt + N` | Khóa khung & Cho phép Bấm xuyên qua (Click-through) |
| `Alt + X` | Xóa chữ đang hiển thị trên màn hình |

*(Bạn hoàn toàn có thể đổi các phím tắt này trong mục Cài đặt).*
